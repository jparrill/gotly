# gotly
Bitly like service for shorten URLs


## Actions

- Run server. (You need to have Docker up and running)
```bash
make redis
make run
```

- Build 
```bash
make build 
```

- Create Redis DDBB
```bash
make redis
```

- Clean Redis DDBB
```bash
make redis-stop
```

- Tests (Nothing yet)
```bash
make tests
```

- Coverage (Nothing yet) 
```bash
make coverage 
```
